    서울앵무새

    RENEWAL DESIGN

    서울앵무새는 트랜디함으로 무장한 힙한 베이커리 카페입니다. 인기에 힘입어 온라인몰도 운영을 시작했지만

    온라인에서는 서울앵무새의 트랜디한 무드가 느껴지지 않아 아쉬웠습니다. 소비자가 서울앵무새에 기대하는 것은 베이커리의

    맛과 품질만이 아닌 소비하며 느낄 수 있는 트랜디함이라고 생각하여 이 부분을 중점으로 리디자인을 진행했습니다.

    Renewal Direction

    MAIN SCREEN

    서울앵무새의 대표메뉴를 제빵하는 영상을 풀 사이즈로 배치하여 서울앵무새 특유의 무드를 느낄 수 있도록 하는 동시에 대표 메뉴를 자연스럽게 노출시킬 수 있도록 했습니다.

    RESPONSIVE SCREENS

    Pc / Tablet / Moblie

    브라우저 사이즈에 대응하기 쉽도록 구조를 심플하게 구성하여 모바일 기준 작업 후, 테블릿, PC에 대응할 수 있도록 설계했습니다.

    PUBLISHINGPROBLOEM & SOLUTION

    Probloem

    • 헤더 영역 position:sticky 적용이 잘 되다가 사라지는 문제 발생.

    • 공지사항을 글자가 옆으로 이동하는 형태로 만들어주고 싶어서 어떤 방법으로 진행해야할지 고민.

    • 모바일, 테블릿, PC 버젼으로 Break Point를 만들어 연결했으나 사이즈별 대응이 어색해 보임.

    • a태그 활성화 되지 않음.

    Solution

    • sticky가 속한 영역에서만 작동하는 특성이 있다는 것을 이해하고 전체 큰영역을 만들어서 해결.

    • css 애니메이션으로 transform: translateX 값과 시간을 조정하여 움직임을 만들어 주었음.

    • Break Point(390px, 440px, 780px, 1200px)를 4번 만들어 최대한 대응이 될 수 있도록 연결.

    • 다른 컨텐츠에 가려져서 a태그가 보여도 선택이 되지 않았던 것! z-index으로 우선순위 높여 해결.
