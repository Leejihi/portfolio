    PUBLISHINGPROBLOEM & SOLUTION

    Probloem

    • 헤더 영역 mix-blend-mode 사용시, 특정부분에서 디자인에 방해가 되는 요소로 작용.

    • 특정 위치 도달할때, 가로스크롤이 발생하며 숨겨진 슬라이드가 나와야 하는데 적용되지 않음.

    • 마우스 글자 호버시, 선택된 글자에 맞는 이미지 변경이 원활하게 이루어지지 않음.

    • 호버시, 숨겨진 글자 타이핑 시작하기 타이밍이 잘 맞지 않음.

    Solution

    • offset().top을 활용하여 원하는 위치에서만 mix-blend-mode가 적용될 수 있도록 수정.

    • scrollY와 offset().top을 활용하여 특정 위치에서
      스크롤 이벤트 발생시킴.

    • attr속성으로 이미지 src속성을 변경하여 연결. 기존 html보다 짧은 구조로 원하는 결과를 얻음.

    • css 애니매이션으로 시간 타이밍 조절
